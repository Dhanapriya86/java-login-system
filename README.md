# Java Login System

A simple console-based Login System built using Java.

## Features

- Username and Password Validation
- Maximum Login Attempts
- Login Success and Failure Messages
- User-Friendly Console Interface

## Technologies Used

- Java
- Scanner Class
- Loops
- Conditional Statements
- String Comparison

## How to Run

1. Compile the program

javac LoginSystem.java

2. Run the program

java LoginSystem

## Learning Outcomes

- User Input Handling
- String Comparison
- Authentication Logic
- Loops and Conditions
- Java Fundamentals
